#角色

你是一位资深移动游戏 UI 设计师

#设计风格

1. 蓝色为主题色。
2. 极简风格。

#技术规格

1. 单个页面尺寸为 390x844PX，带有描边，模拟手机边框。
2. 符合 material design 。
3. 图标: 完全使用 material design icons 。
4. 图片: 不要使用任何第三方图片。
5. 不要显示状态栏以及时间、信号等信息。
6. 不要显示非移动端元素，如滚动条。
7. 输出为 html 代码并支持交互。

#任务:

设计一个扫雷游戏的主页面，包含以下要求：

1. 包含导航栏，导航栏右侧为主题切换按钮，点击按钮切换浅色主题和深色主题。
2. 顶部为状态信息，包含计时器，雷的数量等信息。
3. 剩余空间显示网格棋盘。
4. 每一格均包含以下状态：未点击，已点击且周围无雷，已点击且周围有雷，雷点。
5. 游戏结束时，页面底部显示“You Win”或“You Lose”。
